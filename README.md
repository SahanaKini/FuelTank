A full-stack web application built using Java, MySQL, HTML, CSS, and JavaScript that provides emergency assistance services like fuel delivery, tire repair, and towing for stranded motorists.
RoadRescue is a web-based platform that connects users with nearby service providers for emergency roadside support.
Users can request: Fuel delivery, Tire repair, Tow service

The system also includes: Separate login for service providers, Admin dashboard, Service status tracking.

This project solves real-life travel problems where drivers get stranded due to fuel shortage, punctured tires, or vehicle failures.

Features:
- User Module
Register & Login
Real-time service request
Choose fuel / tire / towing service
View service status

-Service Provider Module
Register service station
Update availability (Fuel / Tow / Tire)
Accept or reject service requests

-Admin Module
View all users
View all service providers
View total requests
Manage platform data

Tech Stack: 
Frontend, HTML, CSS, JavaScript
Backend: Java
Database: MySQL
Tools: MySQL Workbench, VS Code
